# finance - 企业财务信息抓取

抓取和分析企业的财务数据，包括财报、营收、利润、负债等关键指标，支持多期对比和趋势分析。

## 功能特性

### 核心能力
- **财报数据抓取**：获取年报、季报、中报等定期报告
- **关键指标提取**：营收、净利润、资产负债率等核心财务指标
- **多期对比分析**：支持年报季报多期数据对比
- **行业对标分析**：与同行业上市公司进行对比
- **数据可视化**：生成财务趋势图表
- **异常预警**：识别财务指标异常波动

### 支持的财务维度
- 资产负债表指标
- 利润表指标
- 现金流量表指标
- 盈利能力指标
- 偿债能力指标
- 运营效率指标
- 成长能力指标

## 使用方法

### 基础用法
```bash
./fetch.sh "公司名称"
./fetch.sh "贵州茅台"
./fetch.sh "比亚迪"
```

### 高级用法
```bash
# 获取完整财务报表
./fetch.sh "公司名称" --full

# 仅获取利润表
./fetch.sh "公司名称" --type income

# 仅获取资产负债表
./fetch.sh "公司名称" --type balance

# 指定年份（默认最新）
./fetch.sh "公司名称" --year 2023

# 多期对比（最近3年）
./fetch.sh "公司名称" --years 3

# 行业对标
./fetch.sh "公司名称" --benchmark

# 输出 JSON 格式
./fetch.sh "公司名称" --json

# 下载原始财报 PDF
./fetch.sh "公司名称" --download
```

### 脚本参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `公司名称` | 要查询的企业名称 | 必填 |
| `--full` | 获取完整财务报表 | false |
| `--type` | 报表类型：all/income/balance/cashflow | all |
| `--year` | 指定年份 | 最新年份 |
| `--years` | 对比年份数 | 1 |
| `--benchmark` | 行业对标分析 | false |
| `--json` | JSON 格式输出 | false |
| `--download` | 下载财报 PDF | false |

## 数据来源

### 权威数据源
1. **上海证券交易所** - 上市公司公告
2. **深圳证券交易所** - 上市公司公告
3. **巨潮资讯网** - 证监会指定信息披露网站
4. **东方财富** - 财经数据
5. **同花顺** - 金融数据
6. **Wind资讯** - 专业金融数据

### 数据指标说明

#### 盈利能力指标
| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 毛利率 | (营收-成本)/营收 | >40% |
| 净利率 | 净利润/营收 | >15% |
| ROE | 净利润/净资产 | >15% |
| ROA | 净利润/总资产 | >5% |

#### 偿债能力指标
| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 资产负债率 | 总负债/总资产 | <60% |
| 流动比率 | 流动资产/流动负债 | >2.0 |
| 速动比率 | (流动资产-存货)/流动负债 | >1.0 |

#### 运营效率指标
| 指标 | 说明 | 优秀标准 |
|------|------|----------|
| 应收账款周转率 | 营收/应收账款 | 越高越好 |
| 存货周转率 | 营收/存货 | 越高越好 |
| 总资产周转率 | 营收/总资产 | 越高越好 |

## 行业对标分析

### 对标方法
1. 自动识别企业所属行业
2. 获取行业平均指标
3. 计算企业与行业平均的偏差
4. 生成对标分析图表

## 财务预警指标

### 红色预警 (高风险)
- 连续2年净利润为负
- 资产负债率 > 80%
- 经营活动现金流为负
- 应收账款大幅增长 ( > 50%)

### 黄色预警 (中等风险)
- 净利润增速放缓 ( < 行业平均)
- 毛利率持续下降
- 存货周转率下降
- 关联交易占比过高

## 常见问题

**Q: 为什么有些指标显示 N/A？**
A: 部分指标可能因数据未披露或公司类型不适用而缺失

**Q: 如何获取更详细的财务分析？**
A: 使用 `--full` 参数获取完整报告

**Q: 数据更新频率是多少？**
A: 财报发布后 24 小时内更新
