# finance - 企业财务信息抓取

通过搜索引擎抓取企业财务相关公开信息，覆盖财报数据、营收利润、资产负债、业绩动态四个维度。

## 功能特性

- **四维度检索**：财报数据、营收利润、资产负债、业绩动态
- **多关键词交叉搜索**：每个维度使用多个关键词分别搜索
- **结果缓存**：默认缓存 12 小时

## 使用方法

```bash
bash finance/fetch.sh "贵州茅台"
bash finance/fetch.sh "比亚迪"
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `$1` | 公司名称 | 必填 |

## 检索维度与关键词

| 维度 | 搜索关键词 |
|------|-----------|
| 财报数据 | 财报、年报、季报 |
| 营收利润 | 营收、净利润、毛利率 |
| 资产负债 | 资产负债率、总资产、负债 |
| 业绩动态 | 业绩预告、业绩快报、盈利 |

## 数据来源

- **百度搜索** - 各维度财务信息的唯一数据源

## 缓存机制

- 缓存时长：12 小时
- 缓存目录：`~/.cache/wuwenzixun/finance/`

- `curl` - HTTP 请求
- `grep` / `sed` - HTML 文本处理
