# news - 企业新闻抓取

从搜索引擎抓取企业相关新闻标题，支持多源聚合和自动去重。

## 功能特性

- **多源聚合**：同时从百度搜索和搜狗搜索抓取新闻
- **自动去重**：合并多个来源后自动排序去重
- **可控数量**：支持指定返回新闻条数
- **结果缓存**：默认缓存 1 小时，避免重复请求

## 使用方法

```bash
# 基础用法（默认 10 条）
bash news/fetch.sh "腾讯"

# 指定返回条数
bash news/fetch.sh "阿里巴巴" 5
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `$1` | 公司名称 | 必填 |
| `$2` | 返回新闻条数 | 10 |

## 数据来源

- **百度搜索** - 主要新闻来源
- **搜狗搜索** - 补充新闻来源

## 输出示例

```
## 腾讯 - 新闻动态

抓取时间: 2026-02-07 17:00:00

1. 腾讯发布2025年第四季度财报
2. 微信支付推出新功能
3. 腾讯云与某银行达成战略合作

共获取 3 条新闻
```

## 缓存机制

- 缓存时长：1 小时
- 缓存目录：`~/.cache/wuwenzixun/news/`

## 依赖项

- `curl` - HTTP 请求
- `grep` / `sed` - HTML 文本处理
- `python3`（可选）- URL 编码（无 python3 时自动回退到 curl 方式）
