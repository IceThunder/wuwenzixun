# risk - 企业风险信息检索

通过搜索引擎检索企业风险相关信息，覆盖诉讼、处罚、失信、负面舆情四个维度。

## 功能特性

- **四维度检索**：诉讼风险、行政处罚、失信信息、负面舆情
- **多关键词交叉搜索**：每个维度使用多个关键词分别搜索
- **结果缓存**：默认缓存 24 小时

## 使用方法

```bash
bash risk/check.sh "恒大集团"
bash risk/check.sh "瑞幸咖啡"
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `$1` | 公司名称 | 必填 |

## 检索维度与关键词

| 维度 | 搜索关键词 |
|------|-----------|
| 诉讼风险 | 诉讼、被告、起诉 |
| 行政处罚 | 处罚、罚款、违规 |
| 失信信息 | 失信、被执行、老赖 |
| 负面舆情 | 负面、投诉、曝光 |

## 数据来源

- **百度搜索** - 各维度风险信息的唯一数据源

## 输出示例

```
## 恒大集团 - 风险信息检索

检索时间: 2026-02-07 17:00:00

### 诉讼风险

  - 恒大集团被某公司起诉合同纠纷案
  - 恒大地产涉及多起诉讼案件

### 行政处罚

  - 恒大集团因违规被处罚款
  - 住建部门对恒大作出行政处罚

### 失信信息

  - 恒大集团被列入失信被执行人名单

### 负面舆情

  - 恒大集团负面消息持续发酵
  - 多地业主投诉恒大项目停工

### 风险等级评估

综合风险等级: 需结合详细数据进一步评估
建议: 关注上述风险维度中出现的具体事项
```

## 缓存机制

- 缓存时长：24 小时
- 缓存目录：`~/.cache/wuwenzixun/risk/`

## 依赖项

- `curl` - HTTP 请求
- `grep` / `sed` - HTML 文本处理
- `python3`（可选）- URL 编码（无 python3 时自动回退到 curl 方式）
