# risk - 企业风险信息检索

全面检索企业的各类风险信息，包括诉讼、处罚、失信、负面舆情等，构建企业风险画像。

## 功能特性

### 核心能力
- **诉讼信息检索**：查询企业涉及的法律诉讼案件
- **行政处罚查询**：检索监管部门的行政处罚记录
- **失信被执行**：查询失信被执行人信息
- **司法拍卖信息**：检索司法拍卖相关风险
- **负面舆情监测**：抓取和分析网络负面评价
- **风险等级评估**：基于多维度数据评估综合风险

### 支持的查询类型
- 民事诉讼与仲裁案件
- 行政处罚与监管措施
- 失信被执行与限制消费
- 股权冻结与资产查封
- 安全生产事故
- 环保违规记录
- 税务违规信息

## 使用方法

### 基础用法
```bash
./check.sh "公司名称"
./check.sh "恒大集团"
./check.sh "瑞幸咖啡"
```

### 高级用法
```bash
# 完整风险检查（所有维度）
./check.sh "公司名称" --full

# 仅检查诉讼信息
./check.sh "公司名称" --type lawsuit

# 仅检查行政处罚
./check.sh "公司名称" --type penalty

# 仅检查失信记录
./check.sh "公司名称" --type dishonesty

# 包含风险评分
./check.sh "公司名称" --score

# 输出 JSON 格式
./check.sh "公司名称" --json
```

### 脚本参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `公司名称` | 要查询的企业名称 | 必填 |
| `--full` | 完整风险检查 | false |
| `--type` | 检查类型：all/lawsuit/penalty/dishonor/negative | all |
| `--score` | 显示风险评分 | false |
| `--json` | JSON 格式输出 | false |
| `--days` | 搜索时间范围（天） | 365 |

## 数据来源

### 官方数据源
1. **中国执行信息公开网** - 失信被执行、限制消费
2. **中国裁判文书网** - 诉讼案件文书
3. **国家企业信用信息公示系统** - 行政处罚、工商变更
4. **信用中国** - 信用记录
5. **各省市市场监管局** - 地方处罚信息

### 舆情数据源
1. **百度搜索** - 负面关键词搜索
2. **微博** - 社交媒体舆情
3 **黑猫投诉** - 消费者投诉
4. **知乎** - 讨论与评价

## 风险评分体系

### 评分维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 诉讼风险 | 30% | 案件数量、涉案金额、案件趋势 |
| 处罚风险 | 25% | 处罚次数、罚款金额、处罚级别 |
| 失信风险 | 25% | 失信次数、执行状态、金额 |
| 舆情风险 | 20% | 负面文章数、情感倾向、传播范围 |

### 风险等级

| 分数 | 等级 | 颜色 | 说明 |
|------|------|------|------|
| 0-20 | 低风险 | 🟢 | 基本无风险 |
| 21-40 | 较低 | 🟢 | 风险较低 |
| 41-60 | 中等 | 🟡 | 需关注 |
| 61-80 | 较高 | 🟠 | 重点关注 |
| 81-100 | 高风险 | 🔴 | 高危 |

## 缓存机制

- 风险数据默认缓存 24 小时
- 缓存目录：`~/.cache/wuwenzixun/risk/`
- 敏感数据可能需要实时查询

## 依赖项

- `curl` - HTTP 请求
- `jq` - JSON 处理
- `grep` / `sed` / `awk` - 文本处理

## 注意事项

1. **数据时效性**：部分官方数据可能有延迟
2. **信息完整性**：部分地区数据可能不完整
3. **合规使用**：请遵守数据使用相关法规
4. **仅供参考**：风险评估结果仅供参考，不构成投资建议
