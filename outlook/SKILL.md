# outlook - 企业未来展望建议

基于企业当前状况、历史趋势和行业环境，提供企业未来发展的预测分析和建议。

## 功能特性

### 核心能力
- **短期预测**：1-3 个月的业务展望
- **中期展望**：3-12 个月的发展预测
- **长期趋势**：1-3 年的战略预测
- **情景分析**：乐观/基准/悲观情景推演
- **行动建议**：分优先级的行动方案
- **风险预警**：未来可能面临的风险提示

### 分析框架
- PESTEL 分析（政治、经济、社会、技术、环境、法律）
- SWOT 分析（优势、劣势、机会、威胁）
- 五力分析（行业竞争结构）
- 价值链分析

## 使用方法

### 基础用法
```bash
./predict.sh "公司名称"
./predict.sh "小米集团"
./predict.sh "理想汽车"
```

### 高级用法
```bash
# 完整展望报告
./predict.sh "公司名称" --full

# 短期预测（1-3个月）
./predict.sh "公司名称" --short

# 中期展望（3-12个月）
./predict.sh "公司名称" --medium

# 长期趋势（1-3年）
./predict.sh "公司名称" --long

# 情景分析
./predict.sh "公司名称" --scenario

# 包含SWOT分析
./predict.sh "公司名称" --swot

# 包含PESTEL分析
./predict.sh "公司名称" --pestel

# 投资建议
./predict.sh "公司名称" --investment

# 输出JSON格式
./predict.sh "company" --json

# 生成行动计划
./predict.sh "company" --action-plan
```

### 脚本参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `公司名称` | 要分析的企业名称 | 必填 |
| `--full` | 完整展望报告 | false |
| `--short` | 短期预测 (1-3个月) | false |
| `--medium` | 中期展望 (3-12个月) | false |
| `--long` | 长期趋势 (1-3年) | false |
| `--scenario` | 情景分析 | false |
| `--swot` | SWOT分析 | false |
| `--pestel` | PESTEL分析 | false |
| `--investment` | 投资建议 | false |
| `--json` | JSON格式输出 | false |
| `--action-plan` | 生成行动计划 | false |

## 分析方法论

### PESTEL 分析框架

| 维度 | 分析内容 | 评分标准 |
|------|----------|----------|
| Political | 政策稳定性、支持力度 | -5 to +5 |
| Economic | 宏观经济环境、增长预期 | -5 to +5 |
| Social | 消费趋势、人口结构 | -5 to +5 |
| Technological | 技术创新、研发能力 | -5 to +5 |
| Environmental | 环保法规、碳中和政策 | -5 to +5 |
| Legal | 法规完善度、合规成本 | -5 to +5 |

### SWOT 分析维度

- **优势 (Strengths)**：内部竞争优势
- **劣势 (Weaknesses)**：内部竞争劣势
- **机会 (Opportunities)**：外部市场机会
- **威胁 (Threats)**：外部市场威胁

### 情景分析方法

1. **假设构建**：识别关键不确定性因素
2. **情景设计**：设计 3-5 个可能情景
3. **概率评估**：评估各情景发生概率
4. **影响分析**：分析各情景的影响
5. **策略制定**：制定应对策略

## 预测模型

### 短期预测模型
- 移动平均销量预测
- 季节性调整模型
- 市场占有率外推

### 中期预测模型
- 回归分析（营收 vs 销量/价格）
- 市场份额动态模型
- 产能约束模型

### 长期预测模型
- 行业渗透率模型
- 技术采用曲线模型
- 竞争格局演化模型
