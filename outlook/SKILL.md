# outlook - 企业未来展望建议

基于已采集的新闻、风险、财务数据，生成企业短/中/长期展望和行动建议。

## 功能特性

- **三时间维度预测**：短期（1-3个月）、中期（3-12个月）、长期（1-3年）
- **动态风险评级**：根据风险数据量自动调整关注度（高/中等/待评估）
- **行动建议**：提供 5 条具体行动建议
- **结果缓存**：默认缓存 24 小时

## 使用方法

```bash
bash outlook/predict.sh "比亚迪"
bash outlook/predict.sh "小米集团"
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `$1` | 公司名称 | 必填 |

> 注：outlook 模块不直接抓取网络数据，而是读取其他模块的缓存。建议先运行 news、risk、finance 模块后再生成展望。

## 数据来源

- 读取 `~/.cache/wuwenzixun/` 下 news、risk、finance 模块的缓存文件

## 风险关注度评级逻辑

短期预测中的风险关注度根据 risk 模块检索到的条目数动态判定：

| 风险条目数 | 关注度 |
|-----------|--------|
| > 10 条 | 高 |
| 1-10 条 | 中等 |
| 0 条（无数据） | 待评估 |

## 输出示例

```

生成时间: 2026-02-07 17:00:00

数据基础: 新闻数据(是) | 风险数据(是) | 财务数据(是)

### 短期预测 (1-3个月)

- 风险关注度: 中等（已识别 8 条风险信息）
- 建议: 持续跟踪已识别风险，关注新增风险信号

### 中期展望 (3-12个月)

- 行业趋势: 需结合行业报告综合判断
- 竞争态势: 关注同业动态和市场份额变化
- 建议: 关注行业政策变化及监管动向

### 长期趋势 (1-3年)

- 发展空间: 需结合企业战略和行业前景评估
- 风险因素: 宏观经济、政策监管、技术变革
- 建议: 定期进行全面风险评估，建立长效监测机制

### 行动建议

1. 持续监测企业新闻动态和舆情变化
2. 关注行业政策和监管动向
3. 定期更新风险评估（建议每月一次）
4. 建立关键风险指标预警机制
5. 对已识别风险制定应对预案

---
*本展望基于公开信息生成，仅供参考*
```

## 缓存机制

- 缓存时长：24 小时
- 缓存目录：`~/.cache/wuwenzixun/outlook/`

## 依赖项

- `grep` - 统计风险条目数
- 无需网络请求，仅读取本地缓存文件
